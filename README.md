
# Teste do PB para Analista de Dados

Neste teste você foi contratado por uma empresa de bicicletas, e precisa montar uma analise da companhia para o presidente Cleiton.

Cleiton fornece um conjunto de arquivos que o time de dados extraiu do Data Warehouse da companhia. Os arquivos são:
 1. Calendar 
 2. Customer
 3. Product Categories
 4. Product Categories Sales
 5. Product Subcategories
 6. Product
 7. Returns Data
 8. Sales Data 2020
 9. Sales Data 2021
 10. Sales Data 2022
 11. Territory 
 
###  Atividade 1
 Identificar possíveis relacionamentos entre os dados fornecidos.
 
### Atividade 2
Carregar os dados com os tratamentos adequados na ferramenta de visualização

### Atividade 3
Criar métricas como receita, lucro, quantidade de vendas, item mais vendido, vendas por categoria, locais com mais vendas, maiores consumidores e assim por diante, o que você identificar como relevante.

R (receita) = P (preço) x Q (quantidade)
L (lucro) = R (receita) - D (despesas)

### Atividade 4
 Lembre-se é para o Presidente Cleiton, seja criativo na visualização dos dados e na aparência, então capriche! :-)
